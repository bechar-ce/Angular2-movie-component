Need to know About Angular 2

Setup is not that easy of AngularJS 2 as compare to AngularJS 1.X. In AngularJS 1.x you need to add reference of library and you are good to go but AngularJS 2 is dependent on other libraries and it requires some efforts to set up it. In AngularJS 2 no more controllers and $scope also way of Bootstrapping Angular Application is changed

Mobile Oriented With Better Performance

Angular 2 is mobe mobile oriented. Angular 2 is using Hierarchical Dependency Injection System which is major performance booster in Application. Angular 2 implements unidirectional tree based change detection which again increases performance of your web/mobile Application. As per ng-conf meetup, angular 2 is 5 times faster as compared to angular 1.x

Component Based Programming

Just like other javascript framework like ReactJs, AngularJS 2 is also using component based programming. Using component we have less dependent and faster entities. Angular 2 provides more choice for languages.Angular 2.0 means components are the basic building blocks for UI (User Interface) part. In Angular 2.0 there are no controllers concepts like Angular 1.x; create only directives. Improved Dependency Injection with TypeScript Generics.
